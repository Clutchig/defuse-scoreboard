# Defuse Scoreboard

Simple W/L + Kill tracker for Defly.io Defuse mode.

Tracks:
- Wins
- Losses
- Kills
- Session start/stop time + start/stop score

Stats are saved using `localStorage` (persist across refresh and tab close).

---

## Controls

**Start / Stop**
- `Shift + S`
- Click scoreboard
